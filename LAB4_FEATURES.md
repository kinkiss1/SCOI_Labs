# Лабораторная работа 4: Пространственная фильтрация

## Реализованные возможности
- Линейная фильтрация с редактируемыми ядрами нечетного размера.
- Медианная фильтрация с настраиваемой шириной и высотой окна (нечетные значения).
- Зеркальная обработка границ для обоих фильтров.
- Автоматическая генерация ядра Гаусса для произвольной `sigma` и нечетного размера ядра.
- Готовый пресет: `sigma = 3`, размер ядра `13x13`.
- Оптимизированная обработка через `LockBits` и массивы байтов (без цикла `GetPixel` по каждому пикселю).
- Расчет медианы алгоритмом `Quickselect`.

## Особенности
- Ядро вводится текстом: каждая строка — это строка матрицы, значения разделяются пробелами или `;`.
- Для линейной фильтрации доступна опциональная нормализация по сумме коэффициентов ядра.
- Окно фильтрации показывает исходное изображение и предпросмотр до применения к слою.

## Добавленные файлы
- `SpatialFilteringHelper.cs` — алгоритмы фильтрации и генерация ядра Гаусса.
- `FilteringWindow.axaml` — интерфейс параметров фильтрации и предпросмотра.
- `FilteringWindow.axaml.cs` — логика окна фильтрации.
- Интеграция в главное окно: `MainWindow.axaml`, `MainWindow.axaml.cs`.
