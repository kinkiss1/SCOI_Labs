## Градационные преобразования

**Цель лабораторной работы** — реализовать градационные преобразования изображения.

Градационные преобразования:
- выполняются поэлементно
- применяются к каждому пикселю
- зависят только от значения текущего пикселя

Функция преобразования:
s = T(r)

Где:
- r — входная яркость
- s — выходная яркость

Формула:
G(x, y) = T(I(x, y))

---

## Пример

s = r²

Нормировка:
s = 255 * (r / 255)^2

---

## Алгоритм

```c
for y = 0 to height-1
    for x = 0 to width-1
        G(x, y) = 255.0 * (I(x, y) / 255.0)^2;
```

---

## Эффект

- степень > 1 → затемнение  
- корень → осветление  

---

## Гистограммы

Гистограмма — распределение яркости пикселей.

- X — яркость (0–255)
- Y — количество пикселей

---

## Эффекты

- ↑ яркости → вправо  
- ↓ яркости → влево  
- ↑ контраста → расслоение  
- ↓ контраста → кучкование  

---

## Алгоритм гистограммы

```c
int N[256];

for y = 0 to hi-1
    for x = 0 to wi-1
        c = (I(x,y).r + I(x,y).g + I(x,y).b) / 3;
        N[c]++;

max = max(N);
k = hh / max;

for i = 0 to 255
    drawLine(i, hh-1, i, hh-1 - N[i]*k);
```
